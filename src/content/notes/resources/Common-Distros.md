- **REMnux** — Forensics-focused Linux distro. Includes `Volatility`, `INetSim`, and `Oledump.py`. Used primarily for malware analysis, memory forensics, and document analysis.
- **FlareVM** — Windows-based malware analysis environment by Mandiant/FireEye.
