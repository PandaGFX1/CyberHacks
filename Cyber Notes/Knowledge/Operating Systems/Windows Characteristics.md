Account Authentication 

Authentication – Process of determining whether someone is in-face who they declare to be – Must know account's credentials to gain access into an account 

Account Types 

Built-In – Accounts that are AUTOMATICALLY created when an OS is installed. EVERY Windows has at least an Administrator and Guest built-in account. 

- Administrator – User account with the highest level of privileges/permissions. Users with Administrator can install software/devices, change system config settings, as well as manage other user accounts. THIS ACCOUNT CANNOT BE DELETED 
    

- Guest – Intended for TEMPORARY access when an individual user account does not exist. The abilities of a Guest account are GREATLY LIMITED. 
    

Local – Account authenticated by the local machine. Local account credentials are maintained in a local database called the SECURITY ACCOUNTS MANAGER (SAM) 

Domain – Account authenticated by the DOMAIN CONTROLLER. A domain controller is a centralized server in a Windows network that stores domain account credentials in a database called Active Directory.
