- Website to reference for top 1024 common ports: [1024 Most Common Ports](http://www.vmaxx.net/techinfo/ports.htm)

- Ports can be 0-65535. Any port within 0 to 1024 is known as a common, or well-known, port.
- Common Ports:
	- Port 21 (TCP): File Transfer Protocol (FTP)
	- Port 22: Secure Shell (SSH)
	- Port 22: SSH File Transfer Protocol (SFTP)
	- Port 23 (TCP): Telnet
	- Port 25 (TCP): Simple Mail Transfer Protocol (SMTP)
	- Port 53 (UDP/TCP): Domain Name System (DNS)
	- Port 80 (TCP): HyperText Transfer Protocol (HTTP)
	- Port 110 (TCP): Post Office Protocol Version 3 (POP3)
	- Port 143 (TCP): Internet Message Access Protocol (IMAP)
	- Port 443 (TCP): HyperText Transfer Protocol Secure (HTTPS)
	- Port 445: Server Message Block (SMB)
	- Port 465: Simple Mail Transfer Protocol Secure (SMTPS)
	- Port 587: Simple Mail Transfer Protocol Secure (SMTPS)
	- Port 990: File Transfer Protocol Secure (FTPS)
	- Port: 993: Internet Message Access Protocol Secure (IMAPS)
	- Port 995: Post Office Protocol Version 3 Secure (POP3S)
	- Port 3389: Remote Desktop Protocol (RDP)
