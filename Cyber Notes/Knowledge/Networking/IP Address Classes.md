IP Address Classes 

- Class A (0-127) and Class B (128-191)  
    
    - Used for large enterprise networks and ISP's 
        
- Class C (192-223) - Private IP Range (192.168.XX.XX) 
    
    - Used by residential homes and small businesses 
        
- Class D (224-239) 
    
    - Reserved for MULTICAST addressing (not assignable to networks or nodes) 
        
- Class E (240-255) 
    
    - Reserved for experimental use – not seen in network traffic
