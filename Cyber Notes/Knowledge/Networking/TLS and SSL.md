- In the early 1990s Netscape Communications developed Secure Sockets Layer (SSL) for secure communication. They released SSL 2.0 in 1995.
- In 1999 the Internet Engineering Task Force developed Transport Layer Security (TLS). In 2018 TLS had a significant overhaul and TLS 1.3 for released.
- Operates at the transport layer of the OSI model and allows for secure communication.
- Every server (or client) needs to identify itself with a signed TLS certificate.
	- Server administrator creates a Certificate Signing Request (CSR) and submits it to a Certificate Authority (CA). CA verifies and issues a digital certificate.
