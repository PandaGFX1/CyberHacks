- Main Tasks of Defensive Security:
	- Preventing intrusions from occurring
	- Detecting intrusions when they occur and responding properly
	- Related Tasks:
		- User cyber security awareness
		- Documenting and managing assets
		- Updating and patching systems
		- Setting up preventative security devices
		- Setting up logging and monitoring devices
- Roles in Defensive Security:
	- Security Operations Center (SOC)
		- Team of cyber security professionals that monitors the network and detects malicious events.
		- Areas of interest:
			- Vulnerabilities
			- Policy violations
			- Unauthorized activity
			- Network Intrusions
	- Threat Intelligence
		- Threat is any action that can disrupt or harm a system
		- Threat intel collects information to help a company better prepare against potential attackers. They aim to achieve a threat-informed defence.
	- Digital Forensics and Incident Response (DFIR)
		- Application of science to inveistage crimes and establish facts relating to a computer system.
		- Main Focus:
			- File Systems
			- System Memory (Primarily if a program is run in memory and not saved to the disk)
			- System Logs
			- Network Logs
	- Malware Analysis
		- Aims to learn about malicious programs using various means
		- Types of analysis
			- Static analysis
			- Dynamic analysis
	- Incident Response
		- Specifies the methodology that should be followed to handle an attack or break or misconfiguration. Anything bad.
