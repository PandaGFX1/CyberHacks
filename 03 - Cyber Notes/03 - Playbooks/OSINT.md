- Search Operators supported by Google:
	- Refer to: [Advanced Search Operators](https://github.com/cipher387/Advanced-search-operators-list)
	- `"exact phrase"`: Looks for pages with exact words or phrases
	- `site`: Specify the domain name to limit to your search
	- `-`: Omit search results that contain a specific word/phrase.
	- `filetype:`: Finds files of specified types instead of web pages.
- Advanced Search Engines:
	- Shodan:
		- Refer to: [Shodan Query Examples](https://www.shodan.io/search/examples)
		- Search engine for devices connected to the internet. Search for specific types and versions of any device.
		- Focused on devices such as servers, routers, webcams, and IoT devices.
	- Censys:
		- Refer to: [Censys Search Use Cases](https://support.censys.io/hc/en-us/articles/20720064229140-Censys-Search-Use-Cases)
		- Focuses on internet-connected hosts, websites, certificates, and other assets.
		- Use cases: Enumeration domains in use, auditing open ports/services, discovering rogue assets within a network.
	- Have I Been Pwned (HIBP):
		- Tells if an email address has appeared in a leaked data breach and what has been leaked.
